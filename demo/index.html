Getting started

After you download Animotion.css, move animotion.css to your root's CSS directory. Next, just include Animotion's CSS file inside of your tags: 
<link rel="stylesheet" type="text/css" href="css/animotion.css"/>
Then just use the code generated for the specific animotion or add the right class to your elements!
